# Base

This folder focuses on some base models and modules, recording the main idea and structure.

- [AlexNet](alexnet.ipynb) - [ImageNet Classification with Deep Convolutional Neural Networks](https://proceedings.neurips.cc/paper_files/paper/2012/file/c399862d3b9d6b76c8436e924a68c45b-Paper.pdf)
- [VGG](vgg.ipynb) - [Very Deep Convolutional Networks for Large-Scale Image Recognition](https://arxiv.org/pdf/1409.1556)
- [ResNet](resnet.ipynb) - [Deep Residual Learning for Image Recognition](https://arxiv.org/pdf/1512.03385)
- [MobileNet](mobilenet.ipynb) - [MobileNets: Efficient Convolutional Neural Networks for Mobile Vision Applications](https://arxiv.org/pdf/1704.04861)
  - [V2](mobilenetv2.ipynb) - [MobileNetV2: Inverted Residuals and Linear Bottlenecks](https://arxiv.org/pdf/1801.04381)
